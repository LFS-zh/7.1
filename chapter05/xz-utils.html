5.33. Xz-5.0.3

The Xz package contains programs for compressing and decompressing files. It provides capabilities for the lzma and the newer xz compression formats. Compressing text files with xz yields a better compression percentage than with the traditional gzip or bzip2 commands.

Approximate build time: 0.3 SBU
Required disk space: 14 MB

5.33.1. Installation of Xz-Utils

Prepare Xz for compilation: 

./configure --prefix=/tools

Compile the package: 

make

Compilation is now complete. As discussed earlier, running the test suite is not mandatory for the temporary tools here in this chapter. To run the Xz test suite anyway, issue the following command: 

make check

Install the package: 

make install

Details on this package are located in Section 6.44.2, “Contents of Xz.”
